# Lab 01: Navigation and Files

## Objective

Practice basic Linux navigation and file management commands in Ubuntu.

## Environment

* OS: Ubuntu
* Interface: Terminal
* Lab Type: Local practice lab

## Commands Used

* `pwd`
* `ls`
* `ls -la`
* `mkdir`
* `cd`
* `touch`
* `cp`
* `mv`
* `rm`

## Command Notes

### `pwd`

Shows the current working directory.

### `ls`

Lists files and directories in the current location.

### `ls -la`

Lists all files, including hidden files, with detailed information.

### `mkdir linux_lab_test`

Creates a new directory.

### `cd linux_lab_test`

Moves into that directory.

### `touch notes.txt`

Creates an empty file.

### `mkdir files`

Creates a subdirectory.

### `cp notes.txt files/`

Copies the file into the subdirectory.

### `mv notes.txt first_notes.txt`

Renames the file.

### `rm -r linux_lab_test`

Deletes the directory and everything inside it.
